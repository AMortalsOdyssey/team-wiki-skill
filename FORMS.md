# team-wiki-skill Forms

## First Wiki Project Intake

```markdown
请把第一个 Wiki 项目的信息发给我：

1. Git URL 或带分支页面链接：
2. 本地路径：
3. Name：
4. Branch（如果链接已带分支可省略）：
5. Description：
```

## Log Entry

```markdown
- 2026-04-21 | <git config user.name> | 完成某项更新 | [index](path/to/index.md) | [page](path/to/page.md)
```

## Release Change Entry

```markdown
## 1.0.0 · 2026-04-22

- UI 初稿完成
- 对话演绎 Bug 修复和优化

```

## Git Summary Entry

```markdown
## 标题

- 比较范围：base..head
- 提交：feat: example; fix: another change
- 文件：path/a.py, path/b.md
- 摘要：一段精简摘要
```
